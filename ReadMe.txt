Customer Appointment Database Application
    The progrma provides a UI to interact with a database of customers/users/appointments.

Author: David Matthews Email: dmat104@wgu.edu

IDE: IntelliJ 2021.1.3      JDK: Java SE 17.0.1         JavaFX Version: javafx-sdk-17.0.1

Users must login to the program, then the user can choose to manage customers, appointments or reports by selecting
the associated button.

From the manage customer screen, users can add, update, or delete customers from the table.
From the manage appointment screen, users can add, update, or delete appointments from the table.
From the reports screen, users can generate various reports about appointments

My Custom report shows the next upcoming appointment for all customers in the database.
